# ContextForge

**ContextForge** is a local-first desktop devtool for preparing software projects for AI agents.

It scans project structure, builds AI-ready context, generates task packs, creates agent instructions, and helps developers work more safely with Codex, Cursor, Claude Code and other AI coding tools.

## Stack

- Electron
- React
- TypeScript
- Vite
- Tailwind CSS
- Node.js
- Express
- PostgreSQL
- Ollama integration later
